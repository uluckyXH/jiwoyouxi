## 2048 已完成事项

- 标准 4x4 2048 核心玩法已落地。
- 支持滑动棋盘和方向键操作。
- 计分规则使用标准 2048：只有合并才加分，加合并后的数字。
- 合成链为：
  - `2 -> 4 -> 8 -> 16 -> 32 -> 64 -> 128 -> 256 -> 512 -> 1024 -> 2048 -> 4096 -> ...`
- 首次达到 `2048` 弹成功达成弹窗，但不强制结束本局。
- 继续挑战后，不重复弹同一次 2048 达成弹窗。
- 真正失败条件：棋盘满，并且上下左右都没有可合并相邻块。
- 返回大厅确认弹窗已接入，游戏开始后返回会提示。
- 结算弹窗已统一使用 2048 自己的成功/失败弹窗。
- 背景图 `bg.png` 已作为 2048 页面全屏背景接入。
- 顶部文案已去掉 `数字叠叠乐`，避免顶到摄像头区域。
- 顶部 `大厅` 按钮位置已调整，避免挡住背景主角头部。
- 三个分数文本已叠到背景图预留卡片上。
- “蛋块”等旧称呼已移除，统一叫 `数字块`、`本局得分`、`最大数字`、`回大厅`。
