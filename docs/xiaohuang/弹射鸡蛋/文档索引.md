# 弹射鸡蛋文档

本目录专门沉淀弹射鸡蛋的设计、开发方案、素材规划和验收记录。

当前文档：

- `玩法设计.md`：主题、美术、鸡蛋类型、主角互动和玩法目标。
- `玩法交互打磨方案.md`：当前玩法手感问题、交互打磨顺序和边界反馈方案。
- `物理碰撞设计.md`：不做正式关卡前，先打磨碰撞、障碍破坏、目标反馈和星星收集。
- `横屏适配方案.md`：横屏检测、横屏布局和竖屏继续运行方案。
- `横屏进入流程.md`：首次进入横屏提醒和游戏中转竖屏处理。

后续可以继续补：

- `实现计划.md`：代码拆分和迭代顺序。
- `素材计划.md`：素材清单、命名和切图规则。
- `关卡设计.md`：关卡目标、障碍和鸡蛋类型组合。
