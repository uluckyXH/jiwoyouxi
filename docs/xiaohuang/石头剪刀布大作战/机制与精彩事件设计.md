# 石头剪刀布大作战机制与精彩事件设计

日期：2026-07-08
状态：根据现有代码梳理

## 一、分类

现有页面把玩法开关分成两组：

- 游戏机制：上帝之手、地形障碍、缩圈、据点争夺、悬赏头名。
- 精彩事件：黑洞、叛徒、绝地求生、灭霸响指、能量道具、团队道具、我要打十个。

从重构角度看，可以重新分为：

- 玩家干预：上帝之手。
- 地图规则：障碍、缩圈、据点。
- 动态平衡：悬赏、绝地求生、我要打十个。
- 随机事件：黑洞、叛徒、灭霸响指、能量道具。

## 二、上帝之手

开关：`godHand`

行为：

- 只有开启后，触摸战场才会影响单位。
- 底部按钮可在“推开”和“吸引”之间切换。
- 触摸范围约 `130px`。
- 力度随距离衰减，中心最强。
- 推/吸力只改变速度，不直接改变阵营和胜负。

重构建议：

- 用明显的力场圈和拖尾表现，让玩家知道干预范围。
- 保留推/吸切换，但用图标或分段控件降低文字密度。
- 可以限制为少量能量条，避免玩家持续强干预导致自动乱斗失去观战感。

## 三、地形障碍

开关：`obstacles`

行为：

- 手机宽度下生成 `3` 个障碍。
- 更宽屏幕生成 `5` 个障碍。
- 障碍随机位置，宽 `38..82`，高 `28..76`。
- 单位撞到障碍会反弹，速度衰减约 `0.9`。

设计作用：

- 打断直线追击。
- 给弱势方制造逃跑空间。
- 增加局势不可预测性。

重构建议：

- 障碍不要只是矩形块，可以做成鸡窝主题道具，比如木箱、稻草堆、围栏。
- 障碍位置要避免开局压住单位或挡住 UI。

## 四、缩圈

开关：`shrink`

行为：

- 对局开始 `7s` 后启动。
- 战场边界以每秒约 `9px` 的速度内收。
- 最大收缩量是短边的 `28%`。
- 每隔约 `5.2s` 触发一次“缩圈正在推进”提示。
- 渲染层用红色边框强调缩圈。

设计作用：

- 加快后期碰撞频率。
- 避免单位长时间拉扯。
- 与叛徒、黑洞组合时会提高混乱度。

重构建议：

- 缩圈需要明显的安全区边界。
- 单位贴边反弹时应有反馈。
- 可以把缩圈做成“鸡窝围栏收拢”或“舞台灯光收窄”。

## 五、据点争夺

开关：`controlZones`

行为：

- 当前只生成一个中心据点。
- 手机半径 `48px`，宽屏半径 `62px`。
- 每帧统计据点内三方单位数量。
- 如果某一方数量唯一最多，则该方成为据点 owner。
- 据点分数按 `人数 * 2.8 * dt` 增长。
- 分数达到 `100` 且有 owner 时，该方获胜。
- HUD 状态显示据点 owner 和当前分数。

设计作用：

- 从“清场获胜”变成“占目标获胜”。
- 鼓励单位集中到中心，提升观战密度。
- 给弱势方提供非清场翻盘路径。

重构建议：

- 据点模式应作为新版重点模式之一。
- 中心据点要有进度环、归属颜色和争夺状态。
- 平局争夺时应显示“僵持”，而不是静默不涨分。

## 六、悬赏头名

开关：`bounty`

触发条件：

- 总存活单位不少于 `12`。
- 领先方人数占比至少 `54%`。
- 领先方比第二名至少多 `6` 个单位。

行为：

- 领先方成为悬赏头名。
- 非头名阵营额外追击头名单位。
- 非头名单位获得 `1.16` 倍速度系数。
- 如果头名单位被转化，转化者获得奖励：
  - 至少 `1` 层护盾。
  - 暴走约 `4.6s`。
  - 速度放大 `1.22`。

设计作用：

- 抑制滚雪球。
- 让领先方成为全场目标。
- 制造“围剿头名”的戏剧性。

重构建议：

- 悬赏目标必须视觉突出，比如金色通缉圈。
- 触发和解除都要明确提示。
- 奖励最好用粒子或徽章表达，方便玩家理解。

## 七、黑洞

开关：`blackHole`

行为：

- 首次出现时间为开局后 `9..16s`。
- 出现在战场内随机位置。
- 初始核心半径 `24px`，吸引范围 `82px`。
- 范围内单位被吸向中心。
- 单位进入核心区域后死亡，不发生阵营转化。
- 每吞噬一个单位，黑洞半径和吸引范围都放大 `1.3` 倍。
- 半径上限 `116px`，吸引范围上限 `310px`。

设计作用：

- 制造地图危险点。
- 打破阵营数量优势。
- 与缩圈、据点组合时会显著改变路径。

重构建议：

- 黑洞需要预警圈和持续吸入动效。
- 吞噬应有清晰的“消失”表现，避免玩家误以为被转化。

## 八、我们中出了叛徒

开关：`traitor`

触发规律：

- 开局 `10..15s` 内，如果只剩两个阵营，可能触发一次早期叛徒。
- 常规触发从 `20s` 后开始。
- 之后大约每 `10s` 检查一次。
- 每次随机标记 `1` 或 `2` 个存活阵营中的单位。

行为：

- 被选中的单位出现紫色倒计时圈。
- `2s` 后叛变到另一个随机阵营。
- 叛变后短暂暴走约 `0.9s`。

设计作用：

- 打破稳定优势。
- 制造“突然倒戈”的观战事件。
- 在只剩两方时增加反转机会。

重构建议：

- 叛徒不应只靠紫圈，最好有明显身份变化动画。
- 叛变目标阵营可提前用颜色暗示，但不要完全剥夺惊喜。

## 九、绝地求生

开关：`lastStand`

行为：

- 某阵营只剩 `1` 个单位时，有概率触发一次“绝地分裂”。
- 分裂会在源单位附近生成同阵营新单位。
- 某阵营从 `1` 变为 `0` 时，有概率留下复活印记。
- 复活印记 `5s` 后在最后位置复活一个单位。
- 分裂和复活每个阵营各限制一次。

设计作用：

- 防止弱势方瞬间无戏。
- 制造最后一人的英雄时刻。
- 为结果战报提供“逆转”素材。

重构建议：

- 绝地触发要做成强事件，例如慢动作、聚光、特殊音效。
- 复活印记需要倒计时，让玩家期待是否能撑到复活。

## 十、灭霸响指

开关：`thanos`

触发规律：

- 首次登场时间为开局后 `9..18s`。
- 死斗模式下最晚可到 `23s`。
- 登场前 `3s` 给出危险提示。

行为：

- 灭霸覆盖层出现。
- 登场约 `1.65s` 后执行响指。
- 每个阵营随机移除 `floor(当前人数 / 2)`。
- 覆盖层总显示约 `3.9s`。
- 每局只响指一次。

设计作用：

- 将大规模优势方整体削弱。
- 强行改变三方比例。
- 提供强记忆点。

重构建议：

- 如果继续使用该事件，需要确认素材和版权风险。
- 可替换为鸡窝主题事件，比如“鸡神吹哨”“大风刮过”，保留“各阵营减半”的机制。

## 十一、能量道具和团队道具

开关：

- `powerUps`
- `teamPowerUps`

刷新规则：

- 首个道具约开局 `5s` 后出现。
- 后续间隔约 `6.5s`，带 `-1.8s..+2.4s` 随机浮动。
- 场上最多 `4` 个道具。
- 道具存在 `12s` 后过期。

道具类型：

| 类型 | 行为 |
| --- | --- |
| 加速 | 暴走约 `6s`，速度放大 |
| 护盾 | 获得一次转化抵挡 |
| 分裂 | 在附近生成一个同阵营新单位 |

如果开启团队道具，拾取者所在阵营全队获得同类效果。

重构建议：

- 道具图标要明显区分：速度、护盾、分裂。
- 团队道具要有全队连线或范围波纹，否则玩家不容易理解。

## 十二、我要打十个

开关：`tenAgainstOne`

触发条件：

- 场上只剩两个阵营。
- 劣势方人数大于 `0`。
- 优势方人数至少是劣势方 `10` 倍。
- 优势方按原始克制关系能克制劣势方。
- 每秒检查一次，满足条件后有 `10%` 概率触发。
- 每局最多触发一次。

触发行为：

- 先暂停战斗约 `1.5s`。
- 显示英雄覆盖层约 `1.9s`。
- 劣势方进入 `10s` 暴走。
- 暴走期间劣势方速度 `1.5` 倍。
- 劣势方撞到优势方时，不再转化，而是直接击杀优势方。
- 如果优势方被清零，劣势方以“以弱胜强”获胜。
- 如果 `10s` 结束仍未成功，克制关系恢复。

设计作用：

- 当前最明确的高光翻盘事件。
- 有强叙事：最后少数方反杀十倍敌人。
- 适合新版重点打磨。

重构建议：

- 这是最值得保留并强化的事件。
- 应有专门怒气条、镜头强调、冲刺特效和战报标题。
- 触发概率可在新版调成确定触发或模式内保证触发，避免玩家很少看到。
