# 合成大西瓜视觉风格设计

## 风格方向

「合成大西瓜」不做普通水果合成，后续主题改成「鸡窝水果摊」。

画面应该像一个木箱水果摊：水果圆滚滚、像玩具一样带表情，整体保持鸡窝游戏的暖色、轻松、略带嫌弃感的品牌气质。

## 水果设计

从小到大建议是：

```text
小葡萄 -> 蓝莓 -> 橘子 -> 苹果 -> 桃子 -> 哈密瓜 -> 西瓜 -> 巨型鸡窝西瓜
```

当前游戏内部仍保留 0 到 10 共 11 个等级。素材未补齐前，显示名可以先扩展为：

| 等级 | 临时名称 | 说明 |
| --- | --- | --- |
| 0 | 小葡萄 | 最小等级 |
| 1 | 蓝莓 | 小圆果 |
| 2 | 橘子 | 暖色过渡 |
| 3 | 苹果 | 中小等级 |
| 4 | 桃子 | 柔和色 |
| 5 | 哈密瓜 | 中等级 |
| 6 | 小西瓜 | 进入西瓜段 |
| 7 | 西瓜 | 标准西瓜 |
| 8 | 大西瓜 | 大体积 |
| 9 | 鸡冠西瓜 | 戴小鸡冠的过渡终局水果 |
| 10 | 巨型鸡窝西瓜 | 最终目标 |

每个水果都可以有半眯眼表情，和主角鸡的神态统一。正式素材到位前，Canvas 可以先画简单半眯眼线条作为占位。

## 特色元素

- 水果落下时有“咚”的软弹感。
- 合成时出现鸡毛、小星星、果汁粒子。
- 大西瓜可以戴一个小鸡冠。
- 背景是木箱水果摊。

当前阶段先用程序绘制占位，不引入新素材图：

- 圆形水果保持颜色区分。
- 增加半眯眼表情。
- 合成反馈先保留浮动分数，后续补鸡毛、小星星、果汁粒子。
- 终局合成文案改成「巨型鸡窝西瓜！」。

## UI 文案

分数板像一个小木牌，文案使用：

```text
今日瓜分：2330
```

失败文案：

```text
鸡窝被塞满了！
```

建议同步改：

- 游戏标题：`鸡窝水果摊`
- 暂停标题：`水果摊暂停`
- 最高分：`最高瓜分`
- 重新开始按钮：`再摆一摊`
