# 扫雷设计文档

## 风格方向

扫雷主题做成“鸡窝地板排雷”。

棋盘格子避免冷冰冰的系统默认风格，可以设计成：

- 小木板
- 草垫
- 蛋壳砖

整体目标是保留经典扫雷的识别效率，同时让视觉更贴近《鸡窝游戏》的轻松、离线、可爱风格。

## 雷的设计

雷不使用传统炸弹造型，统一称为“鸡雷”。

视觉设计：

- 黑色鸡蛋主体
- 带一小段引线
- 顶部或侧面有红色鸡冠

## 旗子设计

插旗可以使用下列方向之一，后续根据实际 UI 空间选择最清晰的版本：

- 小红旗
- 鸡爪小旗
- 粉色衬衫布片小旗

旗子需要在小尺寸格子中保持高识别度，优先保证插旗状态一眼可见。

## 数字风格

数字使用圆润字体或更圆润的视觉处理，避免太锋利、太系统化。

颜色可以保留经典扫雷区分，但降低饱和度：

- 1：蓝
- 2：绿
- 3：红
- 4：紫
- 5：棕
- 6：青

## 失败效果

失败效果不要血腥，也不要做夸张爆炸。

可以表现为：

- “砰！”
- 一堆鸡毛飞出来
- 主角鸡在旁边露出嫌弃表情

失败文案：

```text
踩到鸡雷了！
```

## 胜利文案

```text
鸡窝安全！
```
