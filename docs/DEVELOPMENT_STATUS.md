## 俄罗斯方块状态

已完成：

- 7-bag 随机队列、3 个 next 预览、幽灵落点、消行计分、等级、锁定延迟。
- 页面 UI 已改成奶油玻璃/HarmonyOS 6 风格：
  - 顶部返回/标题/最高分/暂停
  - 深色棋盘和奶油外框
  - 左右状态卡
  - 底部 4 键控制面板
- 已移除底部逆时针旋转按钮和独立硬降按钮。
- 暂存 Hold 入口已从可点击操作降级为展示，避免触发异常状态。

待复测：

- 最新控制输入改成控制面板统一 `onTouch` 分发：
  - 点按 `←/→/↓/↻`
  - 左右长按 180ms 后每 80ms 连续移动
  - `↓` 长按 350ms 硬降
- 该版本已编译和安装成功，但还缺最后一轮解锁真机操作验证。

已知风险：

- 早期版本中，底部子控件单独 `onClick/onTouch/gesture` 在真机 `uiInput` 下有事件不触发问题，所以改为父面板统一触摸分发。
- 俄罗斯方块目前是重点优化对象，其他小游戏只做过基础进入/交互检查，未做深度打磨。

## 下一步建议

1. 先完成俄罗斯方块最终真机验证，确认四键操作稳定。
2. 建立每个小游戏的 owner 和 issue。
3. 新增游戏按 `docs/ADDING_GAMES.md` 接入，避免改坏大厅和存档层。
4. 每个 PR 至少附一张真机截图和构建结果。
