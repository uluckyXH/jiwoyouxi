## 变更范围

- 扫雷顶部 HUD 的“大厅”入口不再直接退出，局中状态会先弹出退出确认。
- 鸿蒙系统返回键在扫雷局中状态下不再直接回大厅，而是触发同一个退出确认弹窗。
- 退出确认弹窗复用扫雷现有素材：
  - `popup_success_card.png`
  - `button_yellow.png`
  - `button_green.png`
  - 羽毛动效层
- 新增 `MinesweeperImageButton`，扫雷结算弹窗和退出确认弹窗共用同一套图片按钮样式。

## 行为规则

| 场景 | 结果 |
| --- | --- |
| 未开始本局时点击大厅/系统返回 | 直接回大厅 |
| 已开始本局时点击大厅 | 弹出退出确认 |
| 已开始本局时系统返回 | 弹出退出确认 |
| 退出确认弹窗内点击继续排雷 | 关闭弹窗，并恢复原暂停状态 |
| 退出确认弹窗内点击回大厅 | 停止计时并返回大厅 |
| 弹窗已显示时再次系统返回 | 关闭弹窗 |
| 已结算后点击回大厅 | 直接回大厅 |
